Bootstrap

It is a free and open-source front-end framework for rapid website development.

It also includes HTML and CSS based design templates for typography, forms, buttons, tables, navigation, modals, image carousels and many other, as well as optional JavaScript plugins

See all components

Bootstrap was first developed in Twitter and made public on August 2011.

Why Bootstrap?

  • Easy to use: You only need basic HTML & CSS knowledge.
  • Responsive to the core: Bootstrap's responsive CSS adjusts to phones, tablets, and desktops
  • Compatible: It is compatible with all modern browsers (Chrome, Firefox, Internet Explorer, Safari, and Opera)

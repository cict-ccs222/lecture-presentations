What is Tailwind CSS?

  • Utility-first CSS framework
  • Power to Customize Everything
  • Not a UI Kit
  • Responsive Core
    • Best Suited for Complex Style Guides.

Tailwind !== Bootstrap

They are completely different.

Tailwind works by combining multiple CSS classes.

Traditionally, whenever you need to style something on the web, you write CSS.

Why Tailwind?

  • You aren't wasting energy inventing class names for your CSS.
  • Your CSS stops growing.
  • Making changes will be faster and safer.
  • No need to torture yourself in writing heavy CSS code.
